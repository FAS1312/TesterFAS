Modal window test case

Проверка кнопки "Новая задача"
Предусловие: пользователь заходит на страницу _____(url)
Step1: 
  Пользователь нажимает кнопку "Новая задача"
  Ожидаемый результат:
    всплывает модальное окно с заголовком "Создать задачу",
    текстом "Название" и текстовым полем под ним,
    текстом "Описание" и текстовым полем под ним,
    радиокнопками "Высокий", "Средний", "Низкий", по умолчанию выбрана кнопка "Средний"
    неактивным чекбоксом "Выполнено ?",
    кнопками "Отменить", "Сохранить"
    

Проверка Создания новой задачи
Предусловие: 
    пользователь заходит на страницу _____(url),
    пользователь нажимает на кнопку "Новая задача"
Step 1:
    Пользователь вводит данные в поле для ввода "Название"
    Текстовые данные:
        Read some react-native books
    Пользователь вводит данные в поле для ввода "Описание"
    Текстовые данные:
        don't be a lazy chimp
    Пользователь нажимает кнопку "Сохранить"
    Ожидаемый результат:
        Модальное окно закрывается.
        В списке задач появляется задача с названием "Read some react-native books"
        и описание "don't be a lazy chimp" и средним приоритетом
    
Проверка редактирования задачи
Предусловие:
    пользователь заходит на страницу _____(url),
    пользователь нажимает на кнопку "Новая задача"
    Пользователь вводит данные в поле для ввода "Название"
    Текстовые данные:
        Some task
    Пользователь вводит данные в поле для ввода "Описание"
    Текстовые данные:
        Some Description
    Пользователь нажимает кнопку "Сохранить"
Step 1:
    пользователь нажимает на кнопку редактировать рядом с задачей Some Task
    ожидаемый результат:
        всплывает модальное окно с заголовком "Редактировать задачу",
        текстовым полем "Название", в котором введены данные:
            Some Task
        текстоовым полем "Описание", в котором введены данные:
            Some Description
        радиокнопками "Высокий", "Средний", "Низкий", выбрана кнопка "Средний"
        чекбоксом "Выполнено ?",
        кнопками "Отменить", "Сохранить"
Step 2:
    пользователь меняет значение в поле "Название"
    Текстовые данные:
        Other Task
    пользователь нажимает кнопку сохранить
    Ожидаемый результат:
        Модальное окно закрывается.
        В списке задач задача Some Task теперь имеет название Other Task

Step 3:
    пользователь нажимает на кнопку редактировать рядом с задачей Other Task
        пользователь меняет значение в поле "Описание"
    Текстовые данные:
        Other Description
    пользователь нажимает кнопку сохранить
    Ожидаемый результат:
        Модальное окно закрывается.
        В списке задач задача Other Task теперь имеет описание Other Description
        
Step 4:
    пользователь нажимает на кнопку редактировать рядом с задачей Other Task
    пользователь нажимает на радиокнопку "Высокий"
    пользователь нажимает кнопку сохранить
    Ожидаемый результат:
        Модальное окно закрывается.
        В списке задач задача Other Task теперь имеет Высокий приоритет
        
Step 5:
    пользователь нажимает на кнопку редактировать рядом с задачей Other Task
    пользователь нажимает на радиокнопку "Средний"
    пользователь нажимает кнопку сохранить
    Ожидаемый результат:
        Модальное окно закрывается.
        В списке задач задача Other Task теперь имеет Средний приоритет
Step 6:
    пользователь нажимает на кнопку редактировать рядом с задачей Other Task
    пользователь нажимает на радиокнопку "Низкий"
    пользователь нажимает кнопку сохранить
    Ожидаемый результат:
        Модальное окно закрывается.
        В списке задач задача Other Task теперь имеет Низкий приоритет
Step 7:
    пользователь нажимает на кнопку редактировать рядом с задачей Other Task
    пользователь ставит галочку в чекбоксе "Выполнено ?"
    пользователь нажимает кнопку сохранить
    
    Ожидаемый результат:
        Модальное окно закрывается.
        В списке задач у задачи Other Task название и описание зачеркнуты
